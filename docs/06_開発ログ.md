# 開発ログ・意思決定記録

このプロジェクトの立ち上げから現在までの経緯と、主要な意思決定をまとめたもの。

## プロジェクトの目的

- 個人開発（ポートフォリオ + 技術学習）
- ボードゲーム「Gobblet Gobblers」のブラウザ実装
- リポジトリ: https://github.com/tagami0000/Gobblet-Gobblers

## 意思決定サマリー

要件定義の壁打ちで決めた主な方針。詳細は [01_要件定義書](01_要件定義書.md) を参照。

| 項目 | 決定 | 補足 |
|---|---|---|
| プラットフォーム | Web（ブラウザ）・PC専用 | モバイル対応はスコープ外 |
| 対戦形式 | ローカル2人対戦 | AI・オンラインはスコープ外 |
| ルール範囲 | 基本（3×3）+ 拡張（将来検討） | まず3×3版を完成させる |
| 操作方式 | クリック2ステップ（選択→配置先） | ドラッグ&ドロップより実装がシンプル |
| 覆われたコマ | 表示しない | 原作ルール準拠。重なりはリムで示唆 |
| 特殊ルール | コマ移動で相手が3並び→相手の勝ち | 原作ルール準拠で採用 |
| 技術スタック | React + TypeScript + Vite + Tailwind | — |
| プレイヤー名 | 固定（Player 1 / Player 2） | — |
| ドキュメント言語 | 日本語 | — |

## 作業の経緯

### 1. 要件定義
壁打ち形式でプラットフォーム・対戦形式・ルール・技術スタックを確定。

### 2. プロジェクトセットアップ
- Vite + React + TypeScript + Tailwind CSS v4 で初期化
- git 初期化、GitHub リモート登録、初回コミット

### 3. ゲーム実装
- 純粋関数のゲームロジック（`gameReducer` / `findWin` / `canPlace`）
- `useGameState` カスタムフック
- UIコンポーネント（Board / Cell / PlayerArea / GameStatus / PieceCircle）
- 特殊ルール（露出による即時勝利）を実装

### 4. ドキュメント整備
- `README.md` と `docs/` 配下に5種類の設計ドキュメントを作成
- ファイル名は日本語に統一（`01_要件定義書.md` 〜 `05_コンポーネント設計書.md`）

### 5. コマ画像の差し替え対応
- `src/config/pieceImages.ts` を単一の設定ポイントとして用意
- `public/pieces/` に画像を置き、設定に追記すれば CSS の円から画像に差し替え可能

### 6. コードレビュー（バグ修正）
コード精査で見つけた問題を修正。

| 種別 | 内容 |
|---|---|
| バグ | ゲーム終了後も PlayerArea が「手番中」表示のまま → `winner` を考慮 |
| デッドコード | `BoardCell` の未使用 `selection` prop を削除 |
| コメント誤り | 特殊ルールのコメントを正確な表現に修正 |

### 7. デザイン変更（明るいテーマ）
ダーク（slate系）から amber/orange の明るい配色へ刷新。

### 8. UIリッチ化
ポートフォリオとしての見栄えを重視し、大幅に作り込み。

- 立体的なゴブレット（中空カップ）風のコマ表現
- アニメーション（配置時 pop-in / 選択時 bob / 勝利時 win-pulse / 背景ブロブ）
- 勝利ライン検出を座標付きに拡張し、勝った3マスを発光
- 配置先ホバー時のゴーストプレビュー
- 重なり表現（下のコマのリムを覗かせる・中身は隠す）
- 紙吹雪付きの勝利オーバーレイ + 「もう一度あそぶ」
- 木枠風ボード、すりガラス風の手番表示、表示用フォント（Fredoka / M PLUS Rounded 1c）
- ヘッドレス Chrome で初期/配置中/ホバー/勝利の各状態を実機確認

### 9. 表示不具合の修正
手持ちエリアのコマがカードからはみ出して切れていた問題を修正。

- 原因: カードに `overflow-hidden`、かつ手持ちコマがボード用の大サイズのままで幅オーバー
- 対応: 手持ちコマを一回り小さい比率に、カード幅を拡張、`overflow-hidden` を撤去（ヘッダーは自前で角丸）

### 10. GitHub Pages デプロイ
- `.github/workflows/deploy.yml` を追加（main への push でビルド→デプロイ）
- `vite.config.ts` でビルド時のみ `base: '/Gobblet-Gobblers/'`（dev は `/`）
- 公開URL: https://tagami0000.github.io/Gobblet-Gobblers/
- 要手動設定: リポジトリ Settings → Pages → Source を「GitHub Actions」に変更
