# 画面設計書

```
┌─────────────────────────────────────────────────────┐
│                  Gobblet Gobblers                   │  ← タイトル
│                   Player 1 の手番                   │  ← GameStatus
│                                                     │
│  ┌──────────┐  ┌─────────────────┐  ┌──────────┐  │
│  │ Player 1 │  │                 │  │ Player 2 │  │
│  │          │  │  ┌───┬───┬───┐  │  │          │  │
│  │  大 ○ ○  │  │  │   │   │   │  │  │  大 ○ ○  │  │
│  │  中 ○ ○  │  │  ├───┼───┼───┤  │  │  中 ○ ○  │  │
│  │  小 ○ ○  │  │  │   │   │   │  │  │  小 ○ ○  │  │
│  │          │  │  ├───┼───┼───┤  │  │          │  │
│  └──────────┘  │  │   │   │   │  │  └──────────┘  │
│                │  └───┴───┴───┘  │                 │
│                └─────────────────┘                  │
│                    [ リセット ]                      │  ← ResetButton
└─────────────────────────────────────────────────────┘
```

## エリア別仕様

### タイトル

| 項目 | 値 |
|---|---|
| テキスト | `Gobblet Gobblers` |
| スタイル | 白・太字・4xl |

### GameStatus（手番・勝者表示）

| 状態 | 表示内容 | カラー |
|---|---|---|
| Player 1 の手番 | `手番: Player 1` | 黄色（amber-400） |
| Player 2 の手番 | `手番: Player 2` | 青（sky-400） |
| Player 1 の勝利 | `Player 1 の勝利！` | 黄色（amber-400）・大文字 |
| Player 2 の勝利 | `Player 2 の勝利！` | 青（sky-400）・大文字 |

### PlayerArea（プレイヤーエリア）

| 項目 | 仕様 |
|---|---|
| 配置 | ゲームボードの左（Player 1）・右（Player 2） |
| 手番時 | プレイヤーカラーのリング枠で強調表示 |
| 非手番時 | 不透明度を下げて非アクティブを表現 |
| コマ表示 | 大・中・小 の順に縦並び。使用済みはゴースト（点線）表示 |
| ラベル | 各サイズに「大」「中」「小」テキストを表示 |

### GameBoard（ゲームボード）

| 項目 | 仕様 |
|---|---|
| グリッド | 3×3、セル間は 1px ギャップ |
| セルサイズ | 128×128px（w-32 h-32） |
| 外枠 | 角丸（rounded-xl）、スレートグレー背景 |

### BoardCell（各セル）の状態

| 状態 | 背景色 | 枠線色 |
|---|---|---|
| 通常（空） | slate-700 | slate-600 |
| 通常（コマあり） | slate-700 | slate-600 |
| 選択中（ボードから選択） | white/10 | white |
| 配置可能ターゲット | green-900/30 | green-400 |

### ResetButton

| 項目 | 仕様 |
|---|---|
| テキスト | `リセット` |
| スタイル | slate-700 背景、ホバーで slate-600 |

## コマ（PieceCircle）の仕様

### サイズ

| サイズ区分 | クラス | 実寸 |
|---|---|---|
| 大（Large） | `w-20 h-20` | 80px |
| 中（Medium） | `w-13 h-13` | 52px |
| 小（Small） | `w-8 h-8` | 32px |

### カラー

| プレイヤー | 塗り色 | 枠色 |
|---|---|---|
| Player 1 | amber-400（黄色） | amber-600 |
| Player 2 | sky-400（青） | sky-600 |

### 状態別スタイル

| 状態 | スタイル |
|---|---|
| 通常 | 上記カラーで表示 |
| 選択中 | 白いリング（ring-4 ring-white）、1.1倍に拡大 |
| ホバー | 1.05倍に拡大、明度アップ |

## カラーパレット

| 役割 | カラー | Tailwind クラス |
|---|---|---|
| 背景 | ダークネイビー | `bg-slate-900` |
| ボード | ダークグレー | `bg-slate-700` |
| パネル | グレー | `bg-slate-800` |
| グリッド線 | ミドルグレー | `bg-slate-600` |
| Player 1 | ゴールデンイエロー | `amber-400` |
| Player 2 | スカイブルー | `sky-400` |
| 配置可能 | グリーン | `green-400` |
| 選択中 | ホワイト | `white` |

## インタラクションフロー

```
何も選択していない状態
  └─ 自分のコマ（手持ち）クリック  → コマ選択状態へ
  └─ ボード上の自分のコマクリック  → コマ選択状態へ
  └─ その他クリック                → 何も起きない

コマ選択状態
  └─ 配置可能セルクリック          → コマを配置、手番交代
  └─ 配置不可セルクリック          → 何も起きない（選択維持）
  └─ 選択中コマを再クリック        → 選択解除
  └─ 別の手持ちコマクリック        → そのコマに選択変更

ゲーム終了状態
  └─ すべての操作を無効化
  └─ リセットボタンのみ有効
```
