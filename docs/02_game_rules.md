# ゲームルール詳細

## ゲーム概要

Gobblet Gobblers は 2 人用ボードゲームです。  
3×3 のグリッド上で大・中・小のコマを使い、縦・横・斜めのいずれかに 3 つ並べた方が勝ちです。  
大きいコマで相手の小さいコマを「覆う（ゴブル）」ことができ、通常の三目並べより深い戦略が求められます。

## ゲーム構成

### ボード

- 3×3 のグリッド（計 9 マス）

### コマ

各プレイヤーはゲーム開始時に以下のコマを持ちます。

| サイズ | 個数 |
|---|---|
| 大（Large） | 2 個 |
| 中（Medium） | 2 個 |
| 小（Small） | 2 個 |
| **合計** | **6 個** |

## ターンの行動

手番のプレイヤーは以下のどちらか **1 つ** を行います。

### ① 手持ちのコマを置く

手持ちのコマを、ボード上の任意のマスに置きます。

### ② ボード上のコマを移動する

ボード上に置かれた **自分のコマ（最上段のもの）** を、別のマスへ移動させます。

> 移動先にはコマを「覆う」条件（後述）を満たす必要があります。

## コマを覆うルール

コマは **より大きいサイズのコマでのみ覆える** ことができます。

| 覆う側 | 覆われる側 | 可否 |
|---|---|---|
| 大 | 中・小・空 | ✅ 可能 |
| 中 | 小・空 | ✅ 可能 |
| 小 | 空のみ | ✅ 可能（空マスのみ） |
| 同サイズ | 同サイズ | ❌ 不可 |
| 小・中 | 大 | ❌ 不可 |

覆われたコマは **画面上には表示されません**（原作ルール準拠）。

## 勝利条件

縦・横・斜めのいずれかで、**各マスの最上段コマが自分のコマ 3 つになった時点**で勝利です。

```
勝利ライン（計 8 本）

横 3 本:   [0,0][0,1][0,2]
            [1,0][1,1][1,2]
            [2,0][2,1][2,2]

縦 3 本:   [0,0][1,0][2,0]
            [0,1][1,1][2,1]
            [0,2][1,2][2,2]

斜め 2 本: [0,0][1,1][2,2]
            [0,2][1,1][2,0]
```

## 特殊ルール：コマ移動による露出勝利

ボード上のコマを移動させた瞬間に、**下に隠れていたコマが露出**します。  
このとき、露出したコマにより **いずれかのプレイヤーの 3 並びが成立した場合、そのプレイヤーの即時勝利**となります。

### 具体例

```
移動前:
  [0,0] = [相手-小, 自分-大]  ← 自分の大コマが相手の小コマを覆っている
  [0,1] = [相手-中]
  [0,2] = [相手-中]

自分が [0,0] の大コマを別のマスへ移動 → [0,0] の頂上が「相手-小」に変わる

移動後の頂上:
  [0,0] = 相手-小
  [0,1] = 相手-中
  [0,2] = 相手-中
  → 相手が横一列 3 並び → 相手の勝ち
```

> 自分のコマを動かしたとしても、結果的に相手が有利になる場合があります。  
> コマを持ち上げる前に、下に何があるかよく考えることが重要な戦略要素です。
